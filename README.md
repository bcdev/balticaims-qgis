# BalticAIMS QGIS plugin

A plugin for [QGIS](https://qgis.org/) to load [Xcube](https://github.com/xcube-dev/xcube) datasets from an xcube server.
The plugin is being developed for the BalticAIMS project.

## Planned features

- [ ] Represent spatiotemporal datasets with one time step per band and automatically tag bands with time stamps for QGIS' built-in [Temporal Controller](https://docs.qgis.org/3.40/en/docs/user_manual/map_views/map_view.html#time-based-control-on-the-map-canvas).
- [ ] Load (subsets of) data sets from the BalticAIMS server on-demand.
